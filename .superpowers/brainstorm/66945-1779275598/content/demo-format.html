Van's Auto — Backend Demo

First, let's nail down what kind of demo this is. What format best fits the goal?

A

Clickable Mockup / Guided Tour

A polished static demo with fake data baked in. Walks Van's Auto through a scripted tour — "here's your revenue this month, here's your top services, here's what a work order looks like." No real backend needed. Fast to build, easy to hand off, runs in a browser without setup.

B

Live Demo App with Real Backend

A working web app with a real database, real API, and a dashboard that actually queries data. Demo data is seeded in, but the architecture is production-ready. More impressive technically, but takes longer to build and requires hosting.

C

Hybrid: Static Shell + Live Feeling

HTML/CSS dashboard that looks fully real but is powered by a local JSON file of demo data. Has animated charts, interactive elements, and a guided tour button. Feels like a real app but can be opened as a file — no server needed. Sweet spot between speed and polish.
